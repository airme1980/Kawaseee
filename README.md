# Kawasee

シンプルで高速なワンページ為替換算ツールです。JPY / CNY / USD / EUR の 4 通貨間を即座に換算し、最新レートの更新時刻と参考値ディスクレーマーを表示します。

## 特徴
- 金額入力と通貨選択だけで即時換算
- ワンタップで変換元/先を入れ替え
- 4 通貨カードを 2 列で表示し、各カードの金額欄に直接入力
- 入力した通貨を基準に他の 3 通貨の金額欄へ即時反映
- Frankfurter API から取得した最新レートと更新時刻を表示
- レート取得失敗時もユーザーに状況を案内
- モバイル優先のレスポンシブ UI

## 開発・プレビュー
```
# 依存は不要。単純に静的ホスティングしてください。
cd src
python3 -m http.server 5173
```
ブラウザで `http://localhost:5173` を開くと確認できます。任意の静的サーバー（`npx serve`, `busybox httpd` など）でも構いません。

## デプロイ
Azure Static Web Apps などの静的ホスティングで `src/` ディレクトリをそのまま公開してください。追加のビルドステップやバックエンドは不要です。
